# FnoHydro Pretrained Models

The `.pt` (TorchScript) model weight files are **not stored in git** due to their size.

Download them from the JETSCAPE-FNO repository or from the project data release:
https://github.com/JETSCAPE/JETSCAPE-FNO/tree/PythonTest/fno_hydro/models

Place the `.pt` files in this directory before running FnoHydro.
